# Java Pattern Programs

This folder contains Java pattern programs implemented using nested loops, conditions, numbers, characters, and symbols.

## Concepts Used

- Nested Loops
- if-else
- ASCII Values
- Character Manipulation
- Mathematical Logic

## Programs Included

- Star Patterns
- Number Patterns
- Alphabet Patterns
- Binary Patterns
- Diagonal Patterns
- Zig-Zag Patterns

These programs are useful for improving logic-building and preparing for coding interviews.
